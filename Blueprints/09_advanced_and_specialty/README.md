# Phase 9: Advanced and specialty

**Goal:** **Optional** deeper topics: market microstructure (order flow, imbalance, spread), tail dependence (copulas), regime and memory (Hurst, entropy), sizing (Kelly), price models, and stress testing. Use these when a specific problem demands them.

**Why it matters:** You don’t need these to run a solid backtest. But when you need to model fat tails together (copulas), measure toxicity or impact (order flow, spread), or stress-test under crashes, these tests document the tools.

**What you’ll learn:** One page per topic: what it is, when to use it, how it fits with the rest of the suite. Pick and choose; no need to do all of them.

**Tests in this phase:** Hurst exponent, entropy, copulas/tail dependence, correlation breakdown, factor exposure and risk decomposition → order flow toxicity, order imbalance, volume profile, spread impact → Kelly criterion, price models, stress testing scenarios → portfolio optimization (GMV, tangency, risk parity, max diversification) → reinforcement learning for option pricing.
