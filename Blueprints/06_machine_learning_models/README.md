# Phase 6: Machine learning models

**Goal:** Go beyond linear baselines with **ML/DL** models—random forest, SVM, kNN, neural nets, LSTM, Transformer—when you have enough data and need nonlinearity or sequence structure.

**Why it matters:** Linear models only capture linear relationships. If your edge is in interactions or long-range dependence, tree-based or neural models may help. These tests define what each model does and when to use it; they still rely on Phase 3 metrics and Phase 4 validation.

**What you’ll learn:** What each model type is (in one page), when to use it, and how it plugs into your pipeline (features in, predictions out, validate with time-series CV). You don’t need to implement from scratch—just know when to reach for which tool.

**Tests in this phase:** TensorFlow/Keras linear regression (bridge to DL) → random forest, gradient boosting (XGBoost, LightGBM, CatBoost), SVM, kNN → feature importance and SHAP → neural network (overview), MLP, LSTM, Transformer.

**Next:** Phase 7 (strategy building blocks) for levels and options, or Phase 8 (pre-live and reporting) when you’re ready to go live.
