# Fibonacci extensions

## What it is

Computes extension levels beyond the swing (e.g. 127.2%, 161.8%) as profit targets or resistance.

## When to use it

To set profit targets. To compare with retracements and Gann. Use in strategy logic.

## How it works

From swing high/low and range, compute extension levels beyond the swing. Output levels per swing. Tools: custom.

## Inputs

Swing data; config for extension ratios.

## Outputs

Extension levels; optionally plots. Location: results/, logs/.

## Related tests

Swing detection; Fibonacci retracements; Gann angles; statistical significance.
