# Phase 3: Performance and risk metrics

**Goal:** Define *how* you measure strategy performance and risk. Every backtest and report will use these.

**Why it matters:** You need a common language: accuracy, MAE, RMSE for predictions; Sharpe, Sortino, Calmar for risk-adjusted returns; drawdowns and VaR/CVaR for “how bad can it get?” Without these, you can’t compare strategies or know if an edge is worth the risk.

**What you’ll learn:** Basic prediction metrics (accuracy, MAE, RMSE, R²), then risk-adjusted return ratios (Sharpe, Sortino, Calmar), then tail risk (drawdowns, VaR, CVaR). You’ll use these in every later phase.

**Tests in this phase:** Basic metrics (accuracy, MAE, RMSE) → mean squared error, R² → Sharpe/Sortino/Calmar → Information ratio, Omega ratio, Ulcer index → drawdowns → VaR and CVaR.

**Next:** Phase 4 (baselines, tuning, validation)—use these metrics to train simple models and to tune/validate without overfitting.
